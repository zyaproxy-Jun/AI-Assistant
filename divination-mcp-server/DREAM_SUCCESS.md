## 🎯 使用方式

### Claude Desktop中使用

```
用户："帮我解析一个梦：我梦见在海上飞翔，突然掉入水中"

Claude会调用：
interpret_dream({
  dream_description: "我梦见在海上飞翔，突然掉入水中",
  emotions: ["恐惧", "自由"],
  language: "zh-CN"
})

返回专业的5段式解析：
## 概述 Overview
## 关键符号 Key Symbols  
## 情绪地景 Emotional Landscape
## 潜在意义 Potential Meanings
## 反思要点 Reflection Points
```
