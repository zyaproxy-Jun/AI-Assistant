## ⚠️ 重要说明

### 关于实际调用

❌ **网页无法直接调用MCP服务器**  
原因: 浏览器安全限制，无法访问本地进程

✅ **实际使用方式**:
1. 通过 Claude Desktop 使用（推荐）
2. 使用命令行工具 `quick-test.js`
3. 编写自定义 MCP 客户端

### 网页的作用

✅ **参数参考** - 了解需要哪些参数  
✅ **功能展示** - 演示系统界面和流程
