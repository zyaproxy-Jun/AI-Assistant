🎯 使用指南

快速开始 (3 步):

  1. 安装依赖
     $ cd divination-mcp-server
     $ npm install

  2. 构建项目
     $ npm run build

  3. 配置 Claude Desktop
     编辑配置文件，添加 MCP 服务器
     重启 Claude Desktop

🔮 技术架构

运行环境:    Node.js 20+
开发语言:    TypeScript 5.x
通信协议:    Model Context Protocol (MCP)
数据库:      无 (纯计算型)

核心依赖:
  ├─ @modelcontextprotocol/sdk  MCP协议支持
  ├─ iztro                       紫微斗数计算
  ├─ lunar-javascript            农历转换
  ├─ axios                       HTTP客户端
  └─ openai                      AI增强(可选)
